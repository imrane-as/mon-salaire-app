# Mon Salaire - version PDF

Application web installable sur iPhone avec export PDF personnalisé "Imrane Asrir".

## Mise à jour
Remplace l'ancien dossier sur Netlify par ce dossier, puis recharge l'application.
Si l'ancien bouton reste affiché, ferme l'application et rouvre-la.

## Export PDF sur iPhone
1. Sélectionne le mois.
2. Appuie sur "Générer le rapport Imrane Asrir".
3. La fenêtre d'impression iOS s'ouvre.
4. Sur l'aperçu, écarte deux doigts pour ouvrir le PDF.
5. Appuie sur Partager puis "Enregistrer dans Fichiers".

Les données restent enregistrées localement dans Safari.
